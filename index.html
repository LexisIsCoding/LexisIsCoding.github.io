TheatreBlogAssessment 

This was a practice assessment I did in college for a class. It's a basic blog posting website made with C#, HTML, JavaScript, and MVC.

One of the biggest challenges during this project was keeping my workflow straight in my head as this was one of the first self-led class projects I had taken on. 
Using GitHub wasn't required for the class but I decided to use it after facing lots of problems at the beginning of the project. I had to start over a few times as after completing one feature, working on the next I would break the previous one and not be sure how to undo my mistake. I also lost track of which features were complete and where the code was for certain features a lot.
GitHub was a perfect fix for these issues as it let me reverse changes that broke the system through version control and track what I had done and why through commit messages.
